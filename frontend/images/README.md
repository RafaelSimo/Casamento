# Fotos do Casamento

Coloque as fotos do casal nesta pasta:

- **hero.jpg** — Foto principal (aparece no topo do site). Recomendado: 1920x1080px ou maior.
- **foto1.jpg** a **foto6.jpg** — Fotos da galeria. Recomendado: 800x800px (quadradas) ou 800x600px.

As fotos serão exibidas automaticamente no site.
Se uma foto não existir, aparecerá um placeholder com o nome do arquivo esperado.
